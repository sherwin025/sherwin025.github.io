Client Side Capstone

ThingCost is an app utilizing CRUD, it was created for our client side capstone project at Nashville Software School. This application analyzes an item a user wants to purchase and allows them to organize, edit, and create a visual representation for the item. It also displays the cost of an item broken down to a users inputted hourly wage. This app should allow users to better analyze purchasing decisions. There is also a community board for members to post announcements or requests.
